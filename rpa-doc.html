RPA PROJECT Versión 1.0.4

Ingeniería de Energreen Automator

Desarrollo de un motor de automatización robótica (RPA) para la gestión masiva de feedback post-venta, integrando Microsoft Excel con protocolos de comunicación externa.

El Problema

Envío manual de encuestas de satisfacción para Energreen Monterrey. Proceso propenso a errores humanos, fatiga del operador y pérdida de datos en el seguimiento.

La Solución RPA

Automatización del ciclo de vida del mensaje: Extracción de datos, sanitización de strings, inyección en shell de Windows y confirmación de envío en tiempo real.

Arquitectura del Proceso

01

Sanitización y Codificación URI

Uso de algoritmos de reemplazo para convertir caracteres especiales y espacios en formato compatible con navegadores (URL Encoding), asegurando que el mensaje no se rompa durante la transferencia.

02

Inyección de Eventos (Shell Scripting)

Implementación del objeto WScript.Shell para ejecutar llamadas al sistema. Esto permite abrir aplicaciones externas sin intervención del usuario.

03

Sincronización de Latencia

Uso de pausas controladas para compensar la carga de la GUI de WhatsApp Desktop, garantizando que los comandos de teclado (SendKeys) se ejecuten solo cuando la ventana esté activa.

Script Principal (VBA)

Sub EnviarWhatsAppFinal()

   ' Definición del motor de ejecución de comandos de Windows

   Dim Shell As Object

   Set Shell = CreateObject("WScript.Shell")


   ' Preparación del mensaje con URL Encoding para protocolos web

   Mensaje = Replace(Mensaje, " ", "%20")


   ' Apertura de la aplicación externa mediante URI Protocol

   URL = "whatsapp://send?phone=" & Celular & "&text=" & Mensaje

   Shell.Run URL


   ' Gestión de sincronización: Espera de 8s para asegurar carga de la UI

   Application.Wait (Now + TimeValue("00:00:08"))

   SendKeys "~", True ' Emulación de tecla Enter (ASCII 13)

End Sub
